# Tap Out

One-tap urge logger. PWA that lives on your home screen. pnfwr.github.io/tapout

Press the button when you feel the impulse to pick up your phone. That's it — a timestamp is logged. The act of recording is the intervention.

## Setup

1. Host these files anywhere static (GitHub Pages, Vercel, etc.)
2. Open the URL in Safari → Share → Add to Home Screen
3. Tap the icon, tap the button, done

## Stack

Single HTML file + manifest + service worker. No frameworks, no backend. Data stays in localStorage.
